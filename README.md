# Good News
## Зависимости
* make
* docker v23.0.3
* docker-compose v2.17.2

## Запуск в dev режиме
В файле .env необходимо задать GID и UID текущего пользователя
~~~
make
make up
~~~

## Завершение работы
~~~
make stop
make rm
~~~
